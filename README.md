# Navi - Gator
(spelled as aligator)

#### Installation

1. Clone the repo
2. `sudo ./install.sh`
3. Add the following function to .bashrc
`sh
    nav() {
      . navi-gator-sh
    }
`

#### Usage

- help       - Print usage.
- ls         - List the available arguments.
- ll         - List the available arguments with more info.
- add        - Add new navigator item.
- remove     - Delete navigator item.
